# Ghid utilizare – TaskBoard

1. Completează titlul taskului și, opțional, prioritatea și scadența.
2. Apasă **Adaugă**.
3. Bifează checkbox-ul pentru a marca taskul ca Done.
4. Folosește câmpul **Caută...** pentru filtrare după text.
5. Din dropdown selectează All/Open/Done.
6. Apasă **Șterge Done** pentru eliminarea tuturor taskurilor finalizate.
