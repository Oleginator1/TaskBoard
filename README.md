# TaskBoard

Mini Task Manager cu separare UI/logică (taskStore) + teste unitare.

## Tehnologii
- HTML5, CSS3
- JavaScript (ES Modules)
- Vite (dev server)
- Vitest (unit tests)

## Funcționalități
- Adăugare task (titlu, prioritate, scadență)
- Marcarea ca Done / Open
- Ștergere task
- Căutare + filtrare după status
- Ștergere în masă a taskurilor Done
- Persistență în LocalStorage
